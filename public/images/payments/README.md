# Payment logos

Add these image files here so they appear on the Payment Options page:

| File          | Recommended size                    | Format |
|---------------|-------------------------------------|--------|
| `klarna.png`  | **280 × 96 px** (or 560 × 192 for 2x) | PNG (transparent) or SVG |
| `affirm.png`  | **280 × 96 px** (or 560 × 192 for 2x) | PNG (transparent) or SVG |
| `afterpay.png`| **280 × 96 px** (or 560 × 192 for 2x) | PNG (transparent) or SVG |

- **Wide wordmark logos:** use **280 × 96 px** (or double for retina).
- **Square/icon logos:** **120 × 120 px** (or 240 × 240) also work; they’ll scale to fit.

Use transparent backgrounds. Download official logos from each provider’s brand/press or merchant resources.
